## 目标

* 启动前端开发服务器，在首页实时查看“立即发现”按钮的 Galaxy 风格效果，并确保交互正常。

## 执行步骤

1. 在前端目录启动开发服务器：

   * 进入 `front/sheep-music`

   * 运行 `npm run dev`（Vite 开发模式，端口 `8001`）
2. 打开预览地址：

   * 访问 `http://localhost:8001/`
3. 页面验证要点：

   * 首页 `Home` 的推荐 Banner 右侧按钮显示为渐变发光的 Galaxy 风格，文本为“立即发现”。

   * 鼠标悬停轻微上移、发光增强；点击后跳转到 `/discover`。

   * 按钮在亮/暗主题下颜色与阴影自适应（使用 `src/styles/theme.css` 变量）。
4. 回归检查：

   * 其他 Element Plus 按钮样式不受影响。

   * 路由与代理仍按 `vite.config.js` 工作（`/api` 代理到 `http://localhost:9000`）。

## 可能问题与处理

* 端口占用：若 `8001` 被占用，暂时改为 `8002` 启动或关闭占用进程。

* 后端未启动：仅验证样式无影响；如需联调推荐区数据，再启动后端于 `9000`。

## 验收标准

* 开发服务器成功启动并可访问。

* GalaxyButton 在首页可见、交互与导航正常。

* 不影响其他页面与组件的现有风格与功能。
