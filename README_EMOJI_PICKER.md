## ⚡ 快速排查

### emoji 选择器不显示
```
1. 清除浏览器缓存
2. F12 检查控制台是否有错误
3. 确认 main.js 有导入语句
```

### 搜索不工作
```
1. 等待数据加载（首次 ~5 秒）
2. F12 → Application → IndexedDB 检查数据是否存在
3. 尝试不同搜索词
```

### 最近使用丢失
```
1. 检查是否为隐私模式（无法用 localStorage）
2. F12 → Application → LocalStorage 检查数据
3. 尝试重新选择 emoji
```
