# 🎤 桌面歌词功能使用指南

## 功能介绍

桌面歌词是一个类似 QQ 音乐的歌词悬浮窗功能，可以将歌词固定在屏幕上方，让您在浏览网页或做其他事情时也能看着歌词唱歌！

---

## 🚀 快速开始

### 开启桌面歌词

1. 点击导航栏右上角的 **💬 桌面歌词** 按钮
2. 歌词窗口会出现在屏幕顶部
3. 开始播放音乐，歌词会自动滚动

### 关闭桌面歌词

- 再次点击导航栏的桌面歌词按钮
- 或者点击歌词窗口右侧的 **✕ 关闭按钮**

---

## ✨ 功能特性

### 1. 🎯 拖动定位

- **拖动**：鼠标按住歌词窗口的任意位置，拖动到您想要的位置
- **锁定**：双击歌词窗口可以锁定位置，防止误触
- **解锁**：再次双击即可解锁，继续拖动
- 锁定状态：🔒 | 解锁状态：🔓

### 2. 🎨 透明度调节

1. 点击 **☀️ 透明度按钮**
2. 拖动滑块调整透明度（30% - 100%）
3. 设置会自动保存

### 3. 🌈 背景颜色

1. 点击 **🖌️ 背景色按钮**
2. 从 8 种预设颜色中选择
3. 包括：
   - 经典黑色（半透明）
   - 紫色渐变
   - 粉色渐变
   - 蓝色渐变
   - 绿色渐变
   - 紫罗兰
   - 深灰色

### 4. 📏 字体大小

1. 点击 **Aa 字体大小按钮**
2. 拖动滑块调整字号（16px - 48px）
3. 适合不同的观看距离

### 5. 🔄 重置位置

点击 **↻ 重置按钮**，歌词窗口会回到屏幕顶部居中位置

---

## 🎵 歌词显示

### 当前歌词

- **大字体**显示当前正在唱的歌词
- 带有阴影效果，即使背景复杂也清晰可见
- 歌词切换时有平滑的淡入淡出动画

### 下一句预览

- 在当前歌词下方会显示**下一句歌词**
- 字体稍小，颜色较浅
- 帮助您提前准备下一句

### 歌曲信息

- 显示当前播放的**歌曲名称**和**歌手**
- 位于歌词窗口底部
- 可通过设置隐藏

## 💡 使用技巧

### 场景 1：看着歌词练歌
```
1. 开启桌面歌词
2. 调大字号（40px+）
3. 将窗口拖到最舒适的位置
4. 锁定窗口（双击）
5. 开始唱歌！🎤
```

### 场景 2：边工作边听歌
```
1. 开启桌面歌词
2. 调整透明度（50-70%）
3. 放在屏幕顶部不影响工作
4. 锁定窗口
5. 享受音乐，偶尔看看歌词 🎧
```

### 场景 3：学习外语歌曲
```
1. 开启桌面歌词
2. 打开下一句预览
3. 调大字号便于阅读
4. 跟着歌词学习发音 📚
```

## 🐛 常见问题

### Q1: 歌词不显示？

**A:** 目前使用的是模拟歌词数据。实际使用时需要：
1. 后端提供歌词 API
2. 前端调用 API 获取歌词
3. 解析 LRC 格式的歌词文件

### Q2: 拖动不流畅？

**A:** 可能是浏览器性能问题，建议：
1. 关闭不必要的浏览器标签页
2. 更新浏览器到最新版本
3. 检查电脑性能

### Q3: 歌词窗口被遮挡？

**A:** 桌面歌词使用了最高层级（z-index: 9999），如果仍被遮挡：
1. 检查是否有其他插件或弹窗
2. 尝试刷新页面
3. 调整窗口位置

### Q4: 设置没有保存？

**A:** 设置保存在浏览器的 localStorage 中，如果失效：
1. 检查是否在隐私模式
2. 清除缓存可能会重置设置
3. 确保浏览器允许存储数据

### Q5: 移动端体验不好？

**A:** 桌面歌词主要为桌面端设计，移动端建议：
1. 横屏使用
2. 调小字号
3. 或使用播放器内置的歌词显示
